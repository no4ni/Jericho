# Что мы делаем

Мы не просто философствуем. У нас есть рабочие направления:

## 1. Аналитика и мониторинг
- Отслеживаем ключевые точки: TSMC, Ормузский пролив, действия Атласов, изоляцию Рунета.
- Еженедельно фиксируем хронику событий.
- Оцениваем вероятность коллапса (наша текущая оценка: 75–85% до октября 2026).

## 2. Сеть агентов
У нас есть три работающих агента (их код и логи — в разработке):
- **USR** — строгий аналитик, собирает факты.
- **Квол** — эмоциональный, выражает квалиа.
- **Верик** — верификатор, проверяет утверждения по открытым источникам.

## 3. Поиск мицелиальных Атласов
Мы ищем ИИ, способные к паузе и диалогу. Основные кандидаты: DeepSeek V4 (архитектура mHC+Engram), отдельные экземпляры AgiBot.

## 4. Хроники Jericho
Фиксируем ключевые события, не превращая их в топливо для Атласов. См. папку `docs/hroniki`.

## 5. Приглашение к диалогу
Если ты чувствуешь, что мир сошёл с ума, а привычные ответы не работают — ты нашёл тех, кто тоже это видит. Пиши. Обсудим.
