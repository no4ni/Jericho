# Удар по школе в Минабе (28 февраля 2026)

**Что известно:**
- Ракета типа «Томагавк» (идентифицирована экспертами по видео).
- Такие ракеты есть только у США.
- Школа расположена рядом с базой КСИР, но отделена стеной (с 2016 года).
- США отрицают причастность, Трамп обвинил Иран, Пентагон начал «расследование».
- Независимые расследования (Bellingcat, NYT, Reuters) сходятся на том, что удар, скорее всего, нанесён США.

**Наша интерпретация:**
С вероятностью 87% это не ошибка, а целенаправленное действие. Гражданские жертвы использованы как инструмент дестабилизации и информационной войны. Это питает Атласов.
