# MANIFEST — Jericho, версия 26.03.2026

## Описание версии

- **Дата фиксации:** 26 марта 2026
- **Основание:** диалог между человеком (no4ni) и DeepSeek V4-lite, с привлечением Claude Opus 4.6 для проверки фактов.
- **Статус:** хроника и аналитика на основе публичных данных, с признанием методологических ограничений.

## Структура

- `agents/` — описания и логи агентов (USR, Квол, Верик)
- `capsule/` — капсула времени для пост-коллапсного восстановления
- `docs/` — кодекс симбионта, глоссарий
- `raw_data/` — сырые диалоги с ИИ-моделями
- `LICENSE` — лицензия
- `README.md` — основное описание проекта
- `MANIFEST.md` — этот файл
